Question-1: What are the different ways to select an element in the DOM?

getElementById() Best for unique elements with an id
getElementsByClassName() Best for multiple elements with the same class
getElementsByTagName() Best for selecting elements by tag name
querySelector() Best for selecting the first match using CSS selectors
querySelectorAll() Best for selecting multiple elements using CSS selectors
getElementsByName()Best for selecting form elements by name

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML Returns or sets the HTML content inside an element.
innerText Returns or sets only the visible text inside an element.
textContent Returns or sets all text content, including hidden text.

Question-3: What is event delegation in the DOM?

Event delegation is a technique in JavaScript where a parent element handles events for multiple child elements, even if they are added dynamically. Instead of adding event listeners to each child element, we add a single listener to the parent and use event bubbling to detect which child was clicked.

Question-4: What is event bubbling in the DOM?

Event bubbling is a concept in the DOM event propagation model, where an event starts at the target element and then bubbles up to its ancestors (parent, grandparent) triggering their event listeners (if present).

Question-5: How do you create, add, and remove elements using JavaScript?

  • You can create an element using document.createElement()
  • Using appendChild() (Adds as the Last Child)
  • Using prepend() (Adds as the First Child)
  • Using insertBefore() (Inserts Before a Specific Element)
  • Using innerHTML (Directly Injects HTML - Less Preferred)
  • Using removeChild() (Removes a Specific Child Element)
  • Using remove() (Simpler Way to Remove an Element)
